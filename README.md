## 📌 주요기능
### 회원 가입
- 이메일 인증(SMTP, Redis)
- 개인 정보 암호화(AES256, BCryptPasswordEncoder)
  
### 로그인 및 인가
- SpringSecurity
- JWT Token: 2개의 토큰(Access, Refresh)으로 관리
- Refresh Token 정보 저장(Redis)

### 상품
- 상품 이미지 저장(AWS S3)
- 판매 가능 여부 업데이트(DB Scheduling)
- 옵션별 재고 캐싱(Redis)
- 상품 검색

### 주문
- 재고 처리(DB Lock, Redis)
- 주문 취소, 반품(DB Scheduling)
- 주문 전/후 내역 확인
  
### 그 외
- 마이페이지(개인정보 업데이트)
- 장바구니
- 위시리스트
